Projects Here are some projects I built and proud of.

Escapebar

A dynamic website collects real-escape-games information in Taiwan, user can post game reviews and get achievements through contributing contents.

Web front-end built with Next.js, React, Material UI. Backend adopted WordPress as headless CMS, Firebase as backend service and used Node.js, Express.js to build Backend For Front-end server and RESTful API. LINE bot built with LINE messaging API, LIFF and Vue.js.

Know More

LARP Bar

A simple game platform allow user to play LARP (Live Action Role Playing) game and interact in realtime. Game master can control game flow from panel and assign items to specific player's package. Maximum 500 players online simultaneously in record.

Front-end was built with React, Chakra UI. Adopted Firebase as serverless backend service, including authentication, realtime database, cloud function, storage, hosting.

LARP Bar can be integrated with online meet software - Gather Town, player can discover items in Gather Town's room space and the item would be passed to player's package automatically.
